# EduShunya 📚

> **Free CBSE Class 9 & 10 Learning Platform**  
> A 100% free educational initiative for students — delivering quality Maths & Science content through YouTube, Notes, and Quizzes.

---

## 🌟 About

**EduShunya** is a free YouTube-based learning platform designed for CBSE Class 9 and Class 10 students. Our mission is to make high-quality education accessible to every student — completely free of cost.

We cover:
- 📐 **Mathematics** — Real Numbers, Linear Equations, Quadratic Equations & more
- 🔬 **Science** — Chemical Reactions, Exploring Secondary Science & more
- 📄 **Notes & PDFs** — Chapter-wise downloadable notes
- 🎯 **Quizzes** — Chapter-wise MCQ practice for exam preparation

## 🚀 Features

- ✅ **Dark Theme** — Deep navy background with neon blue & purple glow effects
- ✅ **Fully Responsive** — Works on mobile, tablet, and desktop
- ✅ **Sticky Navbar** — With hover dropdowns for Class 9 & Class 10 chapters
- ✅ **Glassmorphism UI** — Premium glass-card design system
- ✅ **Lecture Cards** — Direct links to YouTube lectures
- ✅ **Resource Library** — Notes PDFs, PPTs, E-Books & Practice Sheets
- ✅ **Quiz Zone** — Science Quiz, Maths Quiz & Mixed Quiz
- ✅ **Social Links** — Telegram, WhatsApp & Instagram community

## 🖥️ Getting Started

No build tools required! Simply open any HTML file in your browser:

```bash
# Clone the repository
git clone https://github.com/your-username/edushunya.git

# Open the project
cd edushunya

# Open index.html in your browser
start index.html       # Windows
open index.html        # macOS
xdg-open index.html    # Linux
```
